# 🌤️ Daily Task & Weather Dashboard

A single-file Node.js application that combines a productivity tracker with real-time weather updates.

## 🚀 Key Features
- **Weather Integration**: Dynamic temperature updates for your city.
- **Interactive To-Do**: Client-side task management without page reloads.
- **Fast & Minimal**: Zero front-end frameworks—built with pure JS and Express.
